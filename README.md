# BobWich Restaurant Website

A production-ready restaurant website for BobWich - Tanta's favorite sandwich and burger restaurant since 2011.

## Features
- Home page with hero, best sellers, testimonials, and location and hours
- Menu page with gallery and lightbox view
- About page with brand story and values
- Contact page with contact info, form, and map
- Responsive layout and accessible navigation
- Smooth animations and transitions
- SEO meta tags

## Design
- Color palette: Deep Red (#8B1A1A), Cream (#F5E6D3), Dark (#1A1A1A)
- Typography: Outfit (headings), Inter (body)
- Style: Modern, premium, bold, minimal

## Getting Started

Option 1: Direct Open
Open `index.html` in any modern web browser.

Option 2: Local Server
For the best experience, use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (http-server)
npx http-server

# Using PHP
php -S localhost:8000
```

Then navigate to `http://localhost:8000`.
